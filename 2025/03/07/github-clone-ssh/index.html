注意

需要知道ssh key的配置是针对每台主机的！比如我在某台主机上操作git和我的远程仓库，走ssh协议，就需要配置ssh key，放上去的key是当前主机的ssh公钥。那么如果我换了一台其他主机，想要实现无密登录，也就需要重新配置。

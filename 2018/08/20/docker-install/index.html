macOS安装Docker

系统要求

** Docker for Mac ** 要求系统最低为 macOS 10.10.3 Yosemite。如果系统不满足需求，可以安装 ** Docker Toolbox **。

手动下载安装

如果需要手动安装，只需要下载 Docker for Mac。

如同 macOS 其它软件一样，安装也非常简单，双击下载的 .dmg 文件，然后将那只叫 Moby 的鲸鱼图标拖拽到 Application 文件夹即可（其间需要输入用户密码）。
